# Cryptography Algorithms Implementation

## Objective

This project demonstrates three popular cryptographic techniques:

- AES Encryption & Decryption
- RSA Encryption & Decryption
- SHA-256 Hashing

## Tools

- Python
- PyCryptodome
- hashlib

## Files

- aes.py
- rsa.py
- sha256.py

## Installation

pip install -r requirements.txt

## Run

python aes.py

python rsa.py

python sha256.py

✔ AES Encryption
